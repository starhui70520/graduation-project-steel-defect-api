# Steel Surface Defect Detection System

A client-server steel defect detection application with classic and splicing inference modes.

## Features

- Client-server architecture (PySide6 + FastAPI)
- Classic mode: per-image detection
- Splicing mode: multi-image grid composition
- Report generation

## Tech Stack

- Python, PyTorch, Ultralytics YOLO
- FastAPI, PySide6, qfluentwidgets

## Setup

1. Install dependencies from requirements.txt (if available)
2. Download model weights (not included in this repo)
3. Run: 1. Start API: python API.py
2. Run frontend: python 主程序.py

> Model weight files and datasets are not included due to size.
